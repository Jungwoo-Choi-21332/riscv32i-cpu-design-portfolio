# Weekly Process

주차별 학습 및 구현 과정을 정리한 공간입니다. 채용 검토자가 최종 코드만 보는 것이 아니라 CPU 설계가 어떤 단계로 발전했는지 볼 수 있도록 구성했습니다.

## Flow

1. RV32I ISA와 assembly로 instruction format 이해
2. C/toolchain 실습으로 machine code와 memory behavior 확인
3. Verilog/SystemVerilog building block 구현
4. Single-cycle CPU datapath/controller 통합
5. Pipeline register, forwarding, stall/flush를 추가한 pipelined CPU 구현
6. Timer/peripheral software test material 정리
